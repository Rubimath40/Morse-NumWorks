# Morse-NumWorks
This project is to talk with my friends using the Morse code, with the NumWorks'LED. It use the mu module from the alternative Mu OS for NumWorks. For those who can't install the Mu OS because there calculator is locked, I made a small script that can replace the LED by a fill_rect with **Kandinsky**. If you are in that case, you can find it on my repo  or [here](https://my.numworks.com/python/rubimath40/mu). To use this project, you can download the files and send it to your NumWorks using [this site](https://xcas.univ-grenoble-alpes.fr/nw/nws_en.html) (recommended) or using my [NumWorks repo](https://my.numworks.com/python/rubimath40/morse_numworks)
/!\ IF YOU ARE RUNNING A CUSTOM SOFTARE, THE NUMWORKS WEBSITE MAY RESET YOUR CALCULATOR /!\
