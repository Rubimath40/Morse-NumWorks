# What is Morse-NumWorks?
This project is to talk with my friends using the Morse code, with the NumWorks'LED. It use the mu module from the alternative Mu OS for NumWorks. For those who can't install the Mu OS because there calculator is locked, I made a small script that can replace the LED by a fillrect with kandinsky.

# How to install it on your calculator?
**Which OS do you have?**

1) **Epsilon (The default one)**

**/!\ You should not use this method if you are using something else for the risk of a reset**\
If you are in that case, you have to install the 3 scripts. But that's easy because you can find it on the Numworks'website:\
    The main one: https://my.numworks.com/python/rubimath40/morse_numworks\
    alphabet.py:  https://my.numworks.com/python/rubimath40/alphabet\
    mu.py:        https://my.numworks.com/python/rubimath40/mu\

2) **A custom software (Omega/Upsilon/Mu)**

    1. Mu\
    Good you can use the LED and you don't have to install mu.py on your Numworks.

    2. Omega/Upsilon or an other one\
    The same as the default software: you need mu.py but by an other way.

    To install it, you can:\
    - download the files and send it to your NumWorks using that page: https://xcas.univ-grenoble-alpes.fr/nw/nws_en.html (recommended)\
    - Use my repo on the Upsilon workshop:\
    The main one: https://yaya-cout.github.io/Upsilon-Workshop/view/a228a118-7783-46bd-aab6-13491ea048f7\
    alphabet.py:  https://yaya-cout.github.io/Upsilon-Workshop/view/317085bd-c67e-47e0-b8ff-a978c7ccf58f\
    mu.py (only if you don't have mu):\
        https://yaya-cout.github.io/Upsilon-Workshop/view/d6bd8410-e0e2-4245-a8b8-dc1740855f03

# How to use it?

Once you installed all the files, you just have to run the main.py script. It automatically run the main() function and show the first menu. At that point, you just have to read. To exit the menu, press KEY_ZERO.
